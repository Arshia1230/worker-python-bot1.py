# ربات تلگرام مار و پله 🐍🪜

ربات بازی **مار و پله** برای تلگرام، با امکان بازی **آنلاین داخل گروه‌ها** و
بازی **با کامپیوتر** در چت خصوصی. کاملاً فارسی، با **سه سطح** و **پنل ادمین**.

## ویژگی‌ها
- 🎮 بازی آنلاین گروهی (نوبتی) تا ۶ نفر در گروه‌ها و سوپرگروه‌ها
- 🤖 بازی با کامپیوتر در چت خصوصی
- 🎚 سه سطح: **آسان (۳۶ خانه)**، **متوسط (۱۰۰ خانه)**، **حرفه‌ای (۱۴۴ خانه)** — تنها تفاوت، اندازهٔ زمین و تعداد خانه‌هاست
- 🖼 نمایش زمینِ بازی به‌صورت تصویر (مار، نردبان و مهرهٔ بازیکن‌ها)
- 🛠 پنل ادمین: آمار، جدول برترین‌ها، پیام همگانی، حالت تعمیر

## پیش‌نیاز
- پایتون ۳٫۱۰ یا بالاتر

## نصب
```bash
pip install -r requirements.txt
```

## تنظیمات (مهم)
فایل `config.py` را باز کنید و این دو مورد را پر کنید:

1. **توکن ربات**: از [@BotFather](https://t.me/BotFather) یک ربات بسازید و توکن را در `BOT_TOKEN` بگذارید.
2. **آیدی ادمین**: آیدی عددی خود را از [@userinfobot](https://t.me/userinfobot) بگیرید و در `ADMIN_IDS` قرار دهید.

می‌توانید به‌جای ویرایش فایل، از متغیرهای محیطی هم استفاده کنید:
```bash
export BOT_TOKEN="توکن_شما"
```

## اجرا
```bash
python bot.py
```

## افزودن به گروه
1. ربات را به گروه اضافه کنید.
2. برای اینکه ربات بتواند پیام‌های قدیمیِ زمین را پاک کند و چت تمیز بماند،
   آن را **ادمین** کنید و دسترسی «حذف پیام‌ها» را بدهید (اختیاری ولی توصیه‌می‌شود).
3. دستور `/newgame` را در گروه بفرستید، سطح را انتخاب کنید، بقیه با دکمهٔ
   «پیوستن» وارد شوند و بعد «شروع بازی» را بزنید.

> نکته: لازم نیست Group Privacy ربات را خاموش کنید؛ دستورها و دکمه‌ها در هر حالتی کار می‌کنند.

## دستورها
| دستور | کاربرد |
|------|--------|
| `/newgame` | ساخت بازی جدید در گروه |
| `/play` | بازی با کامپیوتر در چت خصوصی |
| `/endgame` | پایان‌دادن به بازی فعلی |
| `/stats` | آمار شما |
| `/admin` | پنل مدیریت (فقط ادمین‌ها) |
| `/help` | راهنما |

## قوانین بازی
- هر بازیکن به‌نوبت تاس می‌اندازد و جلو می‌رود.
- پای نردبان → بالا رفتن 🪜 / سرِ مار → پایین آمدن 🐍
- اگر ۶ بیاید، همان بازیکن یک‌بار دیگر می‌اندازد.
- اگر عدد از خانهٔ آخر بیشتر شود، به همان اندازه برمی‌گردد.
- اولین کسی که به خانهٔ آخر برسد، برنده است.

## ساختار فایل‌ها
```
snake_ladder_bot/
├── bot.py            # هستهٔ ربات و هندلرها
├── config.py         # تنظیمات (توکن، ادمین، سطح‌ها)
├── game.py           # منطق بازی + ساخت زمین + رسم تصویر
├── database.py       # دیتابیس SQLite
├── keyboards.py      # دکمه‌های این‌لاین
├── texts.py          # متن‌های فارسی
├── requirements.txt
└── README.md
```

## نکات فنی
- وضعیت بازی‌های فعال در حافظه نگه‌داری می‌شود؛ با ری‌استارت ربات، بازی‌های نیمه‌تمام پاک می‌شوند (آمار و کاربران در دیتابیس می‌مانند).
- اگر کتابخانهٔ Pillow نصب نباشد، ربات به‌جای تصویر، زمین را به‌صورت متنی نشان می‌دهد و باز هم کار می‌کند.
