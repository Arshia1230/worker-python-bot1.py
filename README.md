# ربات تلگرام مار و پله 🐍🪜

ربات بازی **مار و پله** برای تلگرام، با بازی **آنلاین داخل گروه‌ها** و بازی
**با کامپیوتر در پیوی**. کاملاً فارسی، با **سه سطح**، **پنل ادمین**،
**عضویت اجباری در کانال** و نمایش زمین به‌صورت **جدول متنی خانه‌به‌خانه** (بدون تصویر).

## ویژگی‌ها
- 🎮 بازی با دوستان (نوبتی) تا ۶ نفر داخل گروه‌ها — بدون لینک، مستقیم در همان چت
- 🤖 بازی با کامپیوتر در پیوی (چت خصوصی)
- 🎲 تاس انیمیشنیِ واقعی تلگرام (وقتی تاس می‌اندازید، استیکر تاس می‌چرخد و عدد می‌آورد)
- 🎚 سه سطح: **آسان (۳۶ خانه)**، **متوسط (۱۰۰ خانه)**، **حرفه‌ای (۱۴۴ خانه)** — تنها تفاوت، اندازهٔ زمین است
- 🎨 صفحهٔ بازی به‌صورت جدول ایموجی: عدد فارسی، نردبان 🪜، مار 🐍، خانهٔ پایان 🏁، و بازیکن‌ها با دایره‌های رنگی 🔴🔵🟢🟠🟣🟤
- 🔒 عضویت اجباری در کانال پیش از بازی (Force Join)
- ✨ متن استارت حرفه‌ای، مناسب اشتراک‌گذاری ربات
- 🛠 پنل ادمین: آمار، جدول برترین‌ها، پیام همگانی، حالت تعمیر

## پیش‌نیاز
- پایتون ۳٫۱۰ یا بالاتر

## نصب
```bash
pip install -r requirements.txt
```

## تنظیمات (مهم)
فایل `config.py` را باز کنید (یا متغیرهای محیطی را تنظیم کنید):

1. **توکن ربات** (`BOT_TOKEN`): از [@BotFather](https://t.me/BotFather) بگیرید.
2. **آیدی ادمین** (`ADMIN_IDS`): آیدی عددی خود را از [@userinfobot](https://t.me/userinfobot) بگیرید.
3. **کانال اجباری** (`REQUIRED_CHANNEL`): پیش‌فرض روی `@nii_thv` تنظیم شده.
   - برای **خاموش‌کردن** عضویت اجباری، آن را خالی بگذارید: `REQUIRED_CHANNEL = ""`
   - ⚠️ **بسیار مهم:** ربات باید در آن کانال **ادمین** باشد تا بتواند عضویت کاربران را بررسی کند. اگر ربات ادمینِ کانال نباشد، بررسی عضویت کار نمی‌کند.

## اجرا (روی کامپیوتر خودتان)
```bash
python bot.py
```

## استقرار روی Render و رفع مشکل «خوابیدن» 😴
سرویس‌های **رایگان** Render بعد از حدود **۱۵ دقیقه بی‌فعالیتی** به خواب می‌روند.
این پروژه برای حل این مشکل آماده شده است:

### مراحل استقرار
1. کد را در یک ریپازیتوری GitHub بگذارید.
2. در Render یک **Web Service** بسازید و ریپو را وصل کنید.
3. تنظیمات:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `python bot.py`
4. در بخش **Environment** این متغیرها را اضافه کنید:
   - `BOT_TOKEN` = توکن ربات
   - `ADMIN_IDS` = آیدی عددی شما (مثلاً `123456789`)
   - `REQUIRED_CHANNEL` = `@nii_thv` (یا خالی برای غیرفعال‌کردن)
   - (اختیاری) `RENDER_EXTERNAL_URL` = آدرس عمومی سرویس‌تان (برای Self-Ping)

> ربات یک **وب‌سرور سلامت** روی پورت `$PORT` باز می‌کند؛ به همین خاطر Render آن را به‌عنوان Web Service می‌پذیرد و خطای «No open ports» نمی‌دهد.

### بیدار نگه‌داشتن قطعی (توصیهٔ مهم)
چون Render رایگان با نبودِ ترافیک می‌خوابد، مطمئن‌ترین راه این است که یک
سرویس بیرونی، آدرس سرویس شما را مرتب صدا بزند:
1. در [UptimeRobot](https://uptimerobot.com) (رایگان) یا [cron-job.org](https://cron-job.org) ثبت‌نام کنید.
2. یک Monitor از نوع HTTP بسازید و آدرس عمومیِ سرویس Render خود را بدهید.
3. بازهٔ بررسی را **هر ۵ دقیقه** بگذارید.

با این کار سرویس همیشه بیدار می‌ماند و ربات هیچ‌وقت نمی‌خوابد. ✅

> جایگزین حرفه‌ای: می‌توانید به‌جای Web Service از یک **Background Worker** (پولی) استفاده کنید که اصلاً نمی‌خوابد.

## افزودن به گروه
1. ربات را به گروه اضافه کنید.
2. دستور `/newgame` را بفرستید، سطح را انتخاب کنید، بقیه با «➕ پیوستن» وارد شوند و بعد «▶️ شروع بازی» را بزنید.
3. (اختیاری) برای امکانات بیشتر، ربات را ادمینِ گروه کنید.

## بازی با دوستان (بدون لینک) 👥
بازی با دوستان داخل گروه و به‌صورت مستقیم انجام می‌شود (هیچ لینکی لازم نیست):
1. در پیوی ربات، دکمهٔ «👥 بازی با دوستان (افزودن به گروه)» را بزنید و ربات را به گروه دلخواه اضافه کنید.
2. داخل گروه دستور `/newgame` را بفرستید و سطح را انتخاب کنید.
3. دوستانتان با دکمهٔ «➕ پیوستن» وارد می‌شوند؛ بعد «▶️ شروع بازی» را بزنید.
4. هرکس در نوبت خودش «🎲 پرتاب تاس» را می‌زند و **تاس انیمیشنیِ تلگرام** همان‌جا در گروه می‌چرخد و عدد می‌آورد.

## دستورها
| دستور | کاربرد |
|------|--------|
| `/start` | شروع و منوی اصلی |
| `/newgame` | ساخت بازی جدید در گروه |
| `/play` | بازی با کامپیوتر در پیوی |
| `/endgame` | پایان‌دادن به بازی فعلی |
| `/stats` | آمار شما |
| `/admin` | پنل مدیریت (فقط ادمین‌ها) |
| `/help` | راهنما |

## پنل ادمین
با دستور `/admin` (فقط برای آیدی‌های داخل `ADMIN_IDS`) باز می‌شود و شامل:
- 📊 **آمار**: تعداد کاربران و بازی‌ها بر اساس سطح
- 🏆 **برترین‌ها**: جدول بازیکنان بر اساس برد
- 📢 **پیام همگانی**: ارسال هر پیام به همهٔ کاربران
- 🔧 **حالت تعمیر**: روشن/خاموش‌کردن موقتِ بازی برای کاربران عادی

> اگر `/admin` چیزی نشان نداد، یعنی آیدی شما در `ADMIN_IDS` نیست؛ آن را درست تنظیم کنید.

## قوانین بازی
- هر بازیکن به‌نوبت تاس می‌اندازد و جلو می‌رود.
- پای نردبان → بالا رفتن 🪜 / سرِ مار → پایین آمدن 🐍
- اگر ۶ بیاید، همان بازیکن یک‌بار دیگر می‌اندازد.
- اگر عدد از خانهٔ آخر بیشتر شود، به همان اندازه برمی‌گردد.
- اولین کسی که به خانهٔ آخر برسد، برنده است.
